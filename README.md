# 📊 JobScope India 2026

## AI-Powered Indian Tech Job Intelligence Dashboard

### 📌 Project Overview

JobScope India 2026 is an interactive Streamlit dashboard developed to analyze Indian technology job market trends using a real-world Kaggle dataset.

The dashboard provides insights into hiring patterns, salary trends, experience requirements, and skill domains across major Indian cities.

---

## 🎯 Objectives

- Analyze technology job postings across India.
- Identify top hiring cities and companies.
- Compare salaries across skill domains.
- Analyze experience requirements.
- Provide an interactive dashboard for recruiters, students, and job seekers.

---

## 📂 Dataset

- Source: Kaggle
- Records: 23,201+
- Domain: Indian Technology Jobs (2026)

---

## 🚀 Features

- Interactive Sidebar Filters
- KPI Cards
- AI Insights Panel
- Top Hiring Cities Analysis
- Highest Paying Skill Domains
- Experience vs Salary Analysis
- Fresher-Friendly Companies
- Top Hiring Companies
- Recent Job Listings
- Download Filtered Dataset

---
